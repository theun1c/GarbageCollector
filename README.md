# Garbage Collector Bot 

**Сборщик мусора для твоих мыслей**. Потому что память — дорогой ресурс.

**Ну или потому что мысли — мусор, хз**

**Garbage Collector** — это Telegram-бот, который решает проблему "отправил в Избранное и забыл". Вместо того чтобы терять заметки в недрах Telegram, ты отправляешь их боту, а он автоматически парсит, категоризирует и сохраняет.

## 🛠 Технологии

- **Язык**: Python 3.10+
- **Библиотека**: [Aiogram](https://docs.aiogram.dev/) 
- **База данных**: SQLite
- **Парсинг**: Регулярные выражения + хештеги

## Архитектура

Структура:
```
app/
  bot.py
  config.py
  handlers/
  keyboards/
  services/
  storage/
  models/
```

Ответственности:
- handlers/ принимают сообщения и команды
- keyboards/ строят кнопки и панели тегов
- services/ содержат бизнес-логику: парсинг заметок, назначение тегов, расчет времени напоминаний
- storage/ работает с БД
- models/ описывает сущности: Tag, Note, ReminderRule
