# Garbage Collector Bot 

> Сборщик мусора для твоих мыслей. Потому что память — дорогой ресурс. Ну или потому что мысли - мусор, хз

**Garbage Collector** — это Telegram-бот, который решает проблему "отправил в Избранное и забыл". Вместо того чтобы терять заметки в недрах Telegram, ты отправляешь их боту, а он автоматически парсит, категоризирует и сохраняет.

## 🛠 Технологии

- **Язык**: Python 3.10+
- **Библиотека**: [Aiogram](https://docs.aiogram.dev/) 
- **База данных**: SQLite
- **Парсинг**: Регулярные выражения + хештеги
